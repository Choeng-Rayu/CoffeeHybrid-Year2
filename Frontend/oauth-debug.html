üöÄ Quick Fixes

If "This site can't be reached" error:

  1. Check Backend: Ensure backend server is running on port 5000
  2. Check Network: Try accessing http://localhost:5000/api/health directly
  3. Check CORS: Backend should allow your React app's origin
  4. Check OAuth URL: Should redirect to Google, not show error

If OAuth redirects but doesn't work:

  1. Check Callback URL: Must match Google Console settings
  2. Check Token Processing: React app should handle URL parameters
  3. Check User Context: Login function should be called
